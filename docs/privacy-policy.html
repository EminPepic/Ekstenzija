Swagger Service Tester je Chrome ekstenzija koja omogućava učitavanje Swagger/OpenAPI dokumentacije i pokretanje sigurnosnih i performance testova preko odabranog backend servisa.

Podaci koje prikupljamo

  • URL Swagger/OpenAPI dokumentacije koju korisnik unese.
  • Odabrani endpoint, metoda i generisani test payload-i koji se šalju backend servisu.

Podaci koje NE prikupljamo

  • Ne prikupljamo lične podatke korisnika.
  • Ne koristimo kolačiće niti pratimo korisnike.

Kako se podaci koriste

Podaci se koriste isključivo za izvršavanje testova i prikaz rezultata u ekstenziji. Ekstenzija ne šalje podatke trećim stranama osim backend servisa koji korisnik koristi.

Skladištenje podataka

Ekstenzija može lokalno sačuvati podešavanja (npr. backend URL ili API ključ) u lokalnoj memoriji pregledača. Ti podaci se ne šalju nikome.
