Privacy Policy

Last updated: 2026-03-12

Swagger Service Tester is a Chrome extension that allows loading Swagger/OpenAPI documentation and running security and performance tests through a selected backend service.

Data We Collect

  • The Swagger/OpenAPI document URL provided by the user.
  • The selected endpoint, method, and generated test payloads sent to the backend service.
  • Technical metadata for security and abuse prevention (e.g., IP address and user-agent).

Data We Do NOT Collect

  • We do not collect account profiles, names, emails, or payment information.
  • We do not use analytics or third-party trackers.

How Data Is Used

Data is used solely to execute tests and display results in the extension. The extension does not send data to third parties other than the backend service used by the user.

Data Storage

The extension may locally store settings (e.g., backend URL or API key) in the browser's local storage. This data is not sent to anyone.

The backend sets a short-lived, HttpOnly cookie to manage session tokens required for testing. This cookie is not accessible by page scripts.

Contact

If you have privacy-related questions, please contact the project author.
